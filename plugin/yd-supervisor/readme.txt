== Description ==

your-doctor.jp/media/ 用の独立プラグイン。次の3点を実装する：

1. 監修医師カスタム投稿タイプ（yd_doctor）と関連プロフィールフィールド
2. 記事へのターゲットキーワード記録（REST API経由で読み書き可）
3. AIOSEO連携の構造化データ（MedicalWebPage / reviewedBy）の自動出力

依存：
* Advanced Custom Fields PRO（必須）
* All in One SEO Pack（構造化データ機能利用時に必要）
